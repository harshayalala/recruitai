About RecruitAI

Multi-agent AI recruitment system — built free and open

What is RecruitAI?

RecruitAI is a fully open-source, AI-powered resume screening system. It uses a multi-agent pipeline built with LangGraph to process resumes through 4 specialized AI agents — each performing a specific task in the hiring workflow. The result is a complete candidate evaluation with scores, strengths, weaknesses, red flags, interview questions, and a hiring recommendation — all generated in seconds.

Privacy & Data

RecruitAI does not store any resumes or personal data. Resume text is sent to Groq API for analysis and is not retained. All results are stored only in your browser session. When you close the tab, all data is cleared. No account or signup required.
